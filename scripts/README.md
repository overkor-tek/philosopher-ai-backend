# Scripts

Utility scripts for the Philosopher AI Backend.

## Available Scripts

### backup-database.sh

Creates PostgreSQL database backups with automatic retention.

**Usage:**
```bash
# Basic usage (requires DATABASE_URL env var)
./scripts/backup-database.sh

# Custom backup directory
BACKUP_DIR=/path/to/backups ./scripts/backup-database.sh

# Custom retention (days)
RETENTION_DAYS=30 ./scripts/backup-database.sh
```

**Cron example (daily at 2 AM):**
```bash
0 2 * * * cd /path/to/philosopher-ai-backend && DATABASE_URL="your_url" ./scripts/backup-database.sh >> /var/log/db-backup.log 2>&1
```

**Requirements:**
- PostgreSQL client tools (`pg_dump`)
- `DATABASE_URL` environment variable
